Problem Statement

Adarsh is learning Programming in C++ and today he is writing his first program. The task is very simple: Given two integers A and B, write a program to multiply these two numbers and print the result in a new line.

Input Output Format

The first line contains an integer T, the total number of test cases. Then T lines follow, each line contains two integers A and B.

For each test case, multiply A and B and display the product in a new line.
